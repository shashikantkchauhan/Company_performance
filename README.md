# Company Performance Analysis: Investigating the $1.2M Sales Loss

> **Tools:** Microsoft Excel · Pivot Tables · Data Visualization  
> **Dataset:** 7,000+ sales transactions across 130 countries  
> **GitHub:** [View Repository](https://github.com/shashikantkchauhan/Company_performance)

## 📌 Project Overview

This project performs end-to-end Exploratory Data Analysis (EDA) on over 7,000 sales 
transactions to uncover the root causes of a significant net loss despite high revenue volume.

| Metric | Value |
|---|---|
| Total Revenue | ₹11,12,975 |
| Net Loss | -₹2,06,737 |
| Total Transactions | 7,000+ |
| Total Customers | 160 |
| Countries Covered | 130 |

## 💡 Key Business Insights

1. **Systemic Loss Across All Categories** — All 5 product categories (Books, 
Electronics, Fashion, Home Appliances, Toys) are operating at a loss — indicating 
a structural pricing or cost issue, not an isolated problem.

2. **Toys Category — Biggest Drain** — Toys alone contributes ₹44,475+ to the net 
loss, making it the highest priority for pricing restructuring.

3. **Volume vs. Value Misalignment** — Despite serving 160 customers across 130 
countries, the company loses money on every order — suggesting costs far exceed 
the current pricing model.
